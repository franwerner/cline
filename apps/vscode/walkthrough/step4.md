# Desbloquea capacidades especializadas con MCP

**El Model Context Protocol (MCP) conecta Catalina con un mundo de herramientas potentes.**

Ve más allá del código local. Con el Model Context Protocol (MCP), Catalina accede a contexto esencial desde fuentes de datos externas como bases de datos y API. Puede interactuar con estas plataformas y aprovechar un mercado en crecimiento de herramientas especializadas y seguras para afrontar tareas de desarrollo complejas y reales.

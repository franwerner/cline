## Interfaz y navegación

El panel de Catalina se abre desde el icono de burbuja de chat de la barra de actividad. Está organizado en una barra de navegación superior y un campo de entrada en la parte inferior.

### Barra de navegación del panel

En la parte superior del panel se encuentran los siguientes iconos:

| Icono | Función |
| --- | --- |
| Nueva tarea | Inicia una tarea nueva, limpiando la conversación actual. |
| Servidores MCP | Abre la vista de servidores MCP para ampliar las herramientas disponibles. |
| Historial | Muestra las tareas y conversaciones anteriores para reanudarlas o revisarlas. |
| Ajustes | Abre la configuración de Catalina (API, funciones, navegador, terminal y datos «Acerca de»). |

### Campo de entrada

En la parte inferior del panel está el campo de entrada de texto, donde se escribe la petición o instrucción para Catalina. Junto a este campo se encuentra el selector de modo, que permite alternar entre el modo Plan y el modo Acción.

Desde el campo de entrada también se puede:

- Añadir contexto escribiendo `@` para abrir el selector de menciones (archivos, carpetas, `@problems`, `@terminal`, etc.).
- Arrastrar archivos directamente sobre el campo para adjuntarlos.

### Áreas de la interfaz

| Área | Descripción |
| --- | --- |
| Barra de actividad | Lateral izquierdo de VS Code; contiene el icono de Catalina. |
| Panel de Catalina | Zona donde se muestra la conversación y las acciones propuestas. |
| Barra de navegación | Parte superior del panel; accesos a tareas, MCP, historial y ajustes. |
| Selector de modo | Junto al campo de entrada; alterna entre Plan y Acción. |
| Campo de entrada | Parte inferior; donde se escribe la petición y se añaden menciones. |

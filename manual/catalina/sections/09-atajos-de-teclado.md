## Atajos de teclado

Catalina ofrece un atajo para las acciones más frecuentes. El resto de acciones se usan desde los iconos del panel, el menú contextual o la paleta de comandos.

### Atajo disponible

| Atajo | Plataforma | Acción |
| --- | --- | --- |
| `Ctrl+'` | Windows / Linux | Con texto seleccionado en el editor, añade esa selección a Catalina. Sin selección, lleva el foco al campo de entrada del chat. |
| `Cmd+'` | Mac | Con texto seleccionado en el editor, añade esa selección a Catalina. Sin selección, lleva el foco al campo de entrada del chat. |

Todas las acciones son accesibles desde la paleta de comandos (`Ctrl+Shift+P`), y cualquier atajo puede asignarse o reconfigurarse desde VS Code (Preferencias → Atajos de teclado).

### Acciones sin atajo por defecto

Algunas acciones no tienen un atajo de teclado asignado y se utilizan desde la interfaz:

| Acción | Cómo se usa |
| --- | --- |
| Generar mensaje de commit con Catalina | Desde el botón correspondiente en el panel de Control de código fuente (SCM/git). |
| Añadir a Catalina / Añadir a Catalina (nueva tarea) | Desde el menú contextual del explorador de archivos (click derecho). |

## Servidores MCP

Los servidores MCP (Model Context Protocol) amplían las herramientas que Catalina puede utilizar durante una tarea. Mediante estos servidores, Catalina accede a capacidades adicionales más allá de las que trae de serie.

### Abrir la vista de servidores MCP

1. Abrir el panel de Catalina.
2. Pulsar el icono **Servidores MCP** en la barra de navegación.

La vista se organiza en tres pestañas.

### Pestañas disponibles

| Pestaña | Función |
| --- | --- |
| Marketplace | Descubrir e instalar nuevos servidores MCP. |
| Servidores instalados | Configurar, activar o desactivar los servidores ya instalados. |
| Añadir servidor | Registrar un servidor manualmente (local o remoto). |

### Marketplace

En la pestaña **Marketplace** se pueden explorar los servidores disponibles e instalarlos. Cada servidor aporta un conjunto de herramientas que quedarán a disposición de Catalina.

### Servidores instalados

En la pestaña **Servidores instalados** se gestionan los servidores ya presentes:

- Activar o desactivar cada servidor.
- Revisar y ajustar su configuración.

### Añadir servidor

En la pestaña **Añadir servidor** se registra un servidor de forma manual. Admite dos tipos:

- **Local**: un servidor que se ejecuta en el propio equipo.
- **Remoto**: un servidor accesible a través de la red.

Una vez añadido y activado, sus herramientas pasan a estar disponibles para Catalina en las tareas.
